# Veeam VBR Tape Expired Remove
This script help to unprotect the expired tapes and remove from inventory after confirmed.
It's recommended to use on VTL environments when you need to remove old tapes.

Requirements: 
- Veeam Backup & Replication 12.x or above
- Powershell 7.x
- Tape infraestructure configured

Utilization:
1) Copy the script in the path c:\temp on the server
2) Run the script and answer the questions
3) Check the log file at the same path
